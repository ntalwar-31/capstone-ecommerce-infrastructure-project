# capstone-ecommerce-infrastructure-project
## E-Commerce Infrastructure Project for Ypsilanti Marketing and Print Company
### The goal of this project was to build a website and online presence for Ypsilanti Marketing and Print Company, a local printing company in Ypsilanti, MI. This would allow their company to create efficiencies in their organizational processes as well as optimize their order processing system. APIs like Stripe Payment Processing, Resend, and Azure Blob Storage were also used in this project to implement automation that is integrated within the website.
